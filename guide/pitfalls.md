# Все грабли

## 1. Лимит: 100 запросов/ЧАС (не в секунду)

Ошибка:
```json
{"code":8, "message":"rate quota limit exceed: allowed 100 requests"}
```

Стратегия: 90 запросов максимум (10 про запас). `sleep(0.3)` между вызовами.

## 2. Числа приходят строками

```python
# ❌ Неправильно
r["count"] + other["count"]
# ✅ Правильно  
int(r["count"]) + int(other["count"])
```

## 3. folderId — ровно 20 символов

Копипаст из UI добавляет пробел → 21 символ → ошибка.

```python
folder_id = os.getenv("WORDSTAT_FOLDER_ID").strip()
assert len(folder_id) == 20
```

## 4. Не платите зря

`getRegionsTree` — бесплатно и не считается в лимит. Кэшируйте.
`topRequests` + `dynamics` + `regions` — платно.

## 5. Депозит, не подписка

Вы кладёте 500₽ на счёт. Яндекс списывает по факту запросов. 
Неиспользованные деньги не сгорают. 500₽ хватит на 4-5 месяцев.
